# 智能多語言語音轉錄平台

## 專案概述
完整的AI驅動語音轉錄平台，整合AssemblyAI語音識別和Gemini AI智能分析功能，支援傳統中文轉錄、多人對話識別、情緒分析和會議摘要生成。

## 核心功能

### 語音轉錄
- AssemblyAI高精度語音識別
- 支援多種音頻格式（WebM、M4A、MP3等）
- 大檔案處理（支援180+ MB檔案）
- 自動對話者識別和分段
- 實時進度追蹤

### Gemini AI智能分析
- **會議摘要**: 自動提煉重點、決議和行動項目
- **講者識別**: 分析講者角色、說話風格和參與度
- **情緒分析**: 檢測語氣、情緒變化和衝突程度
- **關鍵資訊提取**: 提取主題、關鍵詞、人名、地點等
- **完整分析**: 一鍵執行所有分析功能

### 用戶管理
- 完整的身份驗證系統
- 角色權限控制（管理員/用戶）
- 個人化轉錄配置
- 使用量追蹤

### 進階功能
- 實時錄音界面（最長3小時）
- Word文檔導出
- AI逐字稿整理優化
- 關鍵字高亮搜索
- 響應式設計

## 技術架構

### 前端 (React + TypeScript)
- React 18 with TypeScript
- Tailwind CSS + shadcn/ui
- TanStack Query for state management
- Wouter for routing

### 後端 (Node.js + Express)
- TypeScript Express server
- Drizzle ORM with PostgreSQL
- Passport.js authentication
- File upload handling

### AI 整合
- **AssemblyAI**: 語音轉錄和基礎分析
- **Gemini 1.5 Flash**: 智能內容分析
- 多層次錯誤處理和重試機制

### 資料庫 (PostgreSQL)
- 用戶管理和權限控制
- 轉錄記錄和元數據
- AI分析結果存儲
- 系統日誌追蹤
